Tentang Kami

Nisbi Versatile Technology adalah Laboratorium Teknologi Disruptif yang berkomitmen untuk menghadirkan perubahan besar dalam bagaimana dunia bekerja melalui teknologi canggih. Kami adalah pusat inovasi yang terus mendorong batas-batas teknologi untuk memberikan solusi yang benar-benar mengubah cara dunia beroperasi.

Inovasi

  • Classifier suara paru-paru untuk deteksi Covid-19
  • API analisa judul berita terindikasi objektifikasi perempuan
  • API text sentiment analysis
  • API news URL sentiment analysis
  • Distorsi Media Monitoring untuk pantau berita terindikasi objektifikasi perempuan

Outstanding Achievement

Tenologi kami memiliki 9.7/10 rating popularitas, 99% service level, dan 759+ pengguna.
